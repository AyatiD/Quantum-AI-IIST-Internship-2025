# Quantum AI Internship Projects

## Overview

This repository contains work completed as part of a Quantum AI internship, focusing on the integration of quantum computing concepts with machine learning techniques. The projects explore fundamental and applied aspects of quantum algorithms, quantum machine learning models, and simulation of quantum systems using modern frameworks.

The implementations are developed using IBM’s Qiskit library, with an emphasis on understanding quantum circuits, variational models, and hybrid quantum-classical approaches.

---

## Key Areas Covered

* Quantum Computing Fundamentals
* Quantum Circuit Design and Simulation
* Quantum Machine Learning (QML)
* Hybrid Quantum-Classical Models
* Optimization and Search Algorithms
